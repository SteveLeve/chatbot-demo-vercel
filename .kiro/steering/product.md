# Product Overview

This is a Vercel AI SDK demo portfolio showcasing RAG (Retrieval-Augmented Generation) implementations. The application demonstrates how to build AI-powered chatbots that use vector search to retrieve relevant context from a knowledge base before generating responses.

## Core Features

- **Basic RAG Chatbot**: Question-answering system using Wikipedia data with semantic search
- **Vector Search**: PostgreSQL with pgvector for similarity-based document retrieval
- **Streaming Responses**: Real-time AI response streaming using Vercel AI SDK
- **Knowledge Base**: Wikipedia articles chunked and embedded for retrieval

## Target Use Cases

- Demonstrating RAG patterns and best practices
- Educational resource for building AI chatbots
- Portfolio of AI SDK capabilities
